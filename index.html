🔧 Firebase Setup

Paste your Firebase credentials. One-time setup — saved in browser.

📋 HOW TO GET CREDENTIALS (5 MIN)
  1. Go to console.firebase.google.com → Add project
  2. Click Web (</>) icon → Register → copy firebaseConfig
  3. Enable Authentication → Email/Password sign-in
  4. Enable Firestore Database → Start in test mode
  5. Go to Authentication → Users → Add User (email + password)
  6. First user to log in becomes Admin automatically
  7. Admin can promote/demote others in the Admin tab
